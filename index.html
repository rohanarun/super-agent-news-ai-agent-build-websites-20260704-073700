How an AI agent builds a website

1. Research & planning

Super browses competitors, docs, and inspiration — similar to how founders already research with ChatGPT or Gemini, but using real browser actions.

2. Generate & edit code

The agent writes HTML, CSS, and JS, opens editors, and iterates — unlike voice-first assistants like Siri.

3. Push to GitHub

Super signs in, creates repos, commits, and pushes code with user approval.

4. Publish & iterate

Deploy to hosting, test live pages, and reuse the computer-use cache so updates are faster next time.
